👜 箱包销售小程序 (Bag Sales Mini Program)
🌟 项目简介
本项目是一个集成了电商购物与个性化定制的综合性箱包销售平台。致力于为用户提供从挑选、DIY 设计、社区分享到社交互动的全方位沉浸式购物体验。项目包含移动端（小程序）与功能完善的后台管理系统。

🛠️ 技术架构
uni-app+Vue3+Django+MySQL等

✨ 核心功能
📱 小程序端
商城基础功能+箱包 DIY 制作+ DIY 草稿箱+社区交流中心+社交互动+智能客服助手🤖


💻 后台管理端

 仪表盘分析 商品管理订单管理 DIY 库管理头条/资讯发布: 发布商城最新活动、品牌动态或定制指南。优惠券发布: 创建营销活动，灵活配置满减、折扣等促销策略。社区监督管理: 审核用户发布的社区笔记与评论，维护健康的社区环境。 用户管理: 查看用户画像、管理用户状态及权限分配。

人工客服系统: 接管智能客服无法处理的复杂问题，提供人工即时回复。
