👜 箱包销售小程序 (Bag Sales Mini Program)
🌟 项目简介
本项目是一个集成了电商购物与个性化定制的综合性箱包销售平台。致力于为用户提供从挑选、DIY 设计、社区分享到社交互动的全方位沉浸式购物体验。项目包含移动端（小程序）与功能完善的后台管理系统。

🛠️ 技术架构
前端框架 (小程序): uni-app, Vue 3

后端框架: Python, Django

数据库: MySQL

其他: WebSockets (用于实时聊天/客服)

✨ 核心功能
📱 小程序端 (客户端)
为普通消费者提供流畅的购物、定制与社交体验：

🛒 商城基础功能: 商品浏览、分类筛选、加入购物车、下单支付。

🎨 箱包 DIY 制作: 提供丰富的素材，支持用户在线对箱包进行个性化定制（例如更换图案、添加挂饰等）。

📦 DIY 草稿箱: 随时保存未完成的设计灵感，支持二次编辑与完善。

📝 社区交流中心: 类似“小红书”的种草社区，支持发布图文笔记，分享穿搭与 DIY 成果。

💬 社交互动: 支持添加好友，并在应用内进行点对点实时聊天。

🤖 智能客服助手: 7x24小时 AI 智能客服，快速解答商品及售前售后疑问。

💻 后台管理端 (Admin)
为运营人员提供高效的数据管理与决策支持工具：

📊 仪表盘分析: 核心业务数据可视化（销售额、订单量、用户增长等）。

🛍️ 商品管理: 商品的增删改查、上下架及库存维护。

📋 订单管理: 订单状态追踪、发货处理及售后退换审核。

🧩 DIY 库管理: 维护和更新小程序前端的 DIY 素材库（图案、配件主图等）。

📰 头条/资讯发布: 发布商城最新活动、品牌动态或定制指南。

🎟️ 优惠券发布: 创建营销活动，灵活配置满减、折扣等促销策略。

🛡️ 社区监督管理: 审核用户发布的社区笔记与评论，维护健康的社区环境。

👥 用户管理: 查看用户画像、管理用户状态及权限分配。

🎧 人工客服系统: 接管智能客服无法处理的复杂问题，提供人工即时回复。
